## 个人简历模板

###  Title

前端求职/程序员求职
[预览地址](https://webkubor.github.io/resume/)

### Intro

- 优化构建，页面秒开无闪烁
- 自适应屏幕兼容移动端
- 可支持部署到线上，可在线浏览
- 自动生成 PDF，全自动化流程
- 支持样式微调
- 生成访问二维码（开发中）
- 颜色优化工具包（开发中）
- 支持英语（开发中）

### Usage
1. fork本项目后再clone到本地修改
2. 进入项目目录执行`pnpm`安装依赖
3. 执行`pnpm dev`打开本地浏览器预览
4. 执行`pnpm pdf `直接生成本地pdf
